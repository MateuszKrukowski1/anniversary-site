# Nim & Tutu Anniversary Website

## How to use
1. Open this folder in VS Code.
2. Add your song as `music.mp3` in the main folder.
3. Add 10 photos in the `images` folder named:
   - photo1.jpg
   - photo2.jpg
   - ...
   - photo10.jpg
4. Open `index.html` with the Live Server extension.

## Customise the text
- Love letter: edit the `letter` text in `script.js`.
- Date reveal: edit the date card text in `index.html`.
- Timeline: edit the timeline text in `index.html`.

## Upload online with GitHub Pages
1. Create a GitHub repository called `anniversary-website`.
2. Upload all files and folders.
3. Go to Settings > Pages.
4. Choose Deploy from branch, branch `main`, folder `/root`.
5. Wait a minute, then use the link GitHub gives you.
